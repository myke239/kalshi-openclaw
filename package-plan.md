# Kalshi OpenClaw Monorepo

Private-first monorepo for three installable packages:

1. `kalshi-plugin` - Kalshi API integration, trading tools, risk gates, persistence, audit.
2. `kalshi-trading-skill` - Safe operating workflow for market analysis, sizing, arming, and trading.
3. `kalshi-strategy-lab` - Experimental simulation, postmortems, and later research workflows.

This repo is scaffolded for iterative development before ClawHub beta release.
